<p align="center">
  A signal-based simulation engine that generates, combines, and animates different waveforms to study interference patterns and waveform behavior.
</p>

<hr>

<h2>Purpose</h2>
<p>
  This project is designed to explore wave mechanics through code by generating different types of signals and analyzing their interactions using constructive and destructive interference.
</p>

<hr>

<h2>Learning Approach</h2>
<ul>
  <li>Understand mathematical representation of waves</li>
  <li>Implement trigonometric signal generation</li>
  <li>Combine signals to observe interference</li>
  <li>Animate signals for dynamic visualization</li>
  <li>Analyze effects of frequency, amplitude, and phase</li>
</ul>

<hr>

<h2>Simulations</h2>
<ul>
  <li><b>Sine Wave</b> — Continuous oscillatory waveform</li>
  <li><b>Cosine Wave</b> — Phase-shifted sine equivalent</li>
  <li><b>Square Wave</b> — Discrete high/low switching signal</li>
  <li><b>Sawtooth Wave</b> — Linear ramp waveform</li>
  <li><b>Constructive Interference</b> — Wave addition</li>
  <li><b>Destructive Interference</b> — Wave subtraction</li>
</ul>

<hr>

<h2>Core Concepts</h2>
<ul>
  <li>Wave superposition principle</li>
  <li>Frequency, amplitude, and phase relationships</li>
  <li>Signal generation using trigonometric functions</li>
  <li>Discrete sampling of continuous signals</li>
  <li>Interference patterns and waveform synthesis</li>
</ul>

<hr>

<h2>Features</h2>
<ul>
  <li>User-defined wave parameters (frequency, amplitude, phase)</li>
  <li>Support for multiple waveform types</li>
  <li>Constructive and destructive interference modes</li>
  <li>Real-time animation using Matplotlib</li>
  <li>Automatic video export of simulations (.mp4)</li>
</ul>
