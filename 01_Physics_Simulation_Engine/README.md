<p align="center">
  A physics-based simulation engine that models and animates fundamental classical systems using numerical methods and real-time visualization.
</p>

<hr>

<h2>Purpose</h2>
<p>
  This project serves as a structured implementation of core physics concepts through code. Each simulation represents a different physical system built from first principles and solved numerically.
</p>

<hr>

<h2>Learning Approach</h2>
<ul>
  <li>Understand the physics behind each system</li>
  <li>Translate equations into code</li>
  <li>Simulate using time-step integration</li>
  <li>Visualize results through animation</li>
  <li>Iterate and refine for accuracy</li>
</ul>

<hr>

<h2>Simulations</h2>
<ul>
  <li><b>Projectile Motion</b> — 2D motion under gravity</li>
  <li><b>Simple Pendulum</b> — Angular oscillatory motion</li>
  <li><b>Spring-Mass System</b> — Harmonic motion using Hooke’s Law</li>
</ul>

<hr>

<h2>Core Concepts</h2>
<ul>
  <li>Time-based simulation (dt stepping)</li>
  <li>Numerical integration (Euler / Semi-Implicit Euler)</li>
  <li>State modeling (position, velocity, acceleration)</li>
  <li>Trigonometry in motion systems</li>
  <li>Oscillatory dynamics</li>
</ul>

<hr>

<h2>Features</h2>
<ul>
  <li>User-defined input parameters</li>
  <li>Real-time animation using Matplotlib</li>
  <li>Multiple physics systems in one program</li>
  <li>Clear separation of logic and visualization</li>
</ul>

<hr>

<h2>Tech Stack</h2>

</ul>
